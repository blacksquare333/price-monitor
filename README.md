# price-monitor 💰
A web scraper that scrapes the data from digital markets

 🧑‍💻Tech stack
- Python
- Selenium
- Supabase
- Pandas

# Features
- Scrapes  data of products that you want
- Stores data in Supabase database
- Ultimately exports data into excel

🔧 Setup
1. Clone the repo with CS code
2. Install dependencies: `pip install -r requirements.txt` 
3. create a table in Supabase for database
4. Run: python scraper.py
